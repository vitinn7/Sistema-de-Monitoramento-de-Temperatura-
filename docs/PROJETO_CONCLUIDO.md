## 🚀 Como Usar o Projeto

### ⚡ Quick Start
```bash
# Clone o repositório
git clone <repository-url>
cd metereologico

# Inicie com Docker (recomendado)
docker-compose up -d

# Acesse as aplicações
# Frontend: http://localhost:3001
# Backend: http://localhost:3000
# Docs API: http://localhost:3000/docs
```

### 🔧 Desenvolvimento Local
```bash
# Backend
cd backend
npm install
npm run dev

# Frontend (em outro terminal)
cd frontend  
npm install
npm start
```

### 🧪 Executar Testes
```bash
# Todos os testes
npm test

# Com coverage
npm run test:coverage

# E2E
npm run test:e2e
```
